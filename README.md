家計簿　　
■作成理由　　
C＋＋でデータベースを操作する練習　　
■想定ユーザー　　
一般人　　
個人経営くらいなら自営業でも使えなくはないと思います　　
■機能一覧　　
・日付、商品、価格、買ったか売ったかを記録　　
・記録したデータを表示　　
■使用技術　　
フレームワーク:VSCODE　　
言語:C＋＋　　
■データ設計の考え方　　
・複数のタグをつけられるようにした　　
・収入を1支出を0で記録できるようにした　　
■工夫点　　
収入と支出の両方を記録できるようにした
